AI Agent Data Analysis & Machine Learning Workbench.

DataAgent Analyst is a portfolio-grade AI data analysis platform built with Python, FastAPI, pandas, scikit-learn, LangGraph, llama.cpp, and vanilla JavaScript.

Users can upload CSV datasets, explore data quality, generate EDA insights, train baseline machine learning models, compare model performance, and generate AI-assisted analysis reports.
