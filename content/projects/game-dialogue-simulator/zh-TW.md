A browser-based simulation game inspired by the Super Wings animated series, using high-quality character art generated from SDXL LoRA models.

- Genre: simulation + dispatch/management - Frontend: Vite + React + TypeScript + Phaser 3 - Backend: FastAPI - Characters available: 8 (Jett, Jerome, Donnie, Chase, Flip, Todd, Paul, Bello)

- Dispatch system: pick a character and send them on missions worldwide - Mission variety: multiple mission types and locations - Resource management: money, fuel, and items - Animation pipeline: launch → flight → arrival → transformation → landing - Persistence: LocalStorage save snapshot for quick resume
