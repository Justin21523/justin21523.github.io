## 專案概述

Vanilla DOM Playground 是一個全功能、零框架的前端學習平台，目標是讓 DOM 操作、事件機制與 CSS 動畫「邊做邊懂」。它由兩種模式組成：可視化 **構建模式**（拖拉元素、設屬性、綁事件、配動畫，並即時生成可運行程式碼）與引導式 **學習模式**（遊戲、教程與實戰項目）。

## 核心功能

構建器提供 Stage 畫布上的拖曳、8 點縮放、智慧輔助線、網格吸附、多選與框選、Z 軸控制等互動；右側面板分為屬性、事件、動畫三個分頁；底部則是 HTML/CSS/JS 即時程式碼預覽。事件系統支援多達 62 種事件類型、8 大分組，並提供 toggleClass、changeColor、setText 等動作範本與 capture/once/passive 進階選項。

## 學習內容

學習模式包含 5 款遊戲（打地鼠、拖放拼圖、貪吃蛇、表單驗證、音樂按鍵）、4 個可視化教程（事件傳播、事件對象、事件委託、自訂事件）與 3 個實戰項目（TODO List、圖片輪播、下拉菜單）。外加一套遊戲化系統：30 個成就、4 種稀有度、進度追蹤、排行榜與 Web Audio 音效。

## 技術架構

全程以純 Vanilla JavaScript 撰寫，約 18,000 行 JS 與近 6,000 行 CSS，採 ES6 模組化，核心包含事件匯流排（Pub/Sub）、集中式狀態管理、命令式歷史與最小化程式碼產生器。專案另附 Docker/Nginx 部署設定與 /docs 下的架構、遊戲、教程與測試文檔。
