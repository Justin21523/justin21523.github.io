CulturePOS Desktop is a C# .NET Avalonia desktop application for bookstores, museum shops, library gift shops, campus co-ops, and cultural retail stores.

> Phase 1 currently includes project structure, login UI, role-based navigation, main shell layout, and mock authentication.

- C# / .NET 10 LTS - Avalonia UI - MVVM - CommunityToolkit.Mvvm - Microsoft.Extensions.DependencyInjection - xUnit for tests

This project is designed for Linux + VS Code.
