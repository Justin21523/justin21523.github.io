## Recommended Length
2-3 minutes.

## Recording Flow
1. Open `https://justin21523.github.io/libracore/`.
2. Show the hero video and explain that GitHub Pages hosts a static demo.
3. Open OPAC search and show the guided assistant spotlight, particles, and step card.
4. Show Discovery facets, bibliographic detail, authority browse, and repository views.
5. Switch to staff workflows: MARC import batch, MARC review, authority control, circulation desk, patron account, acquisitions, serials, ERM, repository, analytics, audit, and data quality.
6. End on GitHub/README links and mention the local setup commands.
